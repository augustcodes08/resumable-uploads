# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Community health files (CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md)
- CHANGELOG.md to track version changes

## [0.1.5] - 2024

### Added

- Support for `chunkRetryInterval` as a function to enable dynamic retry intervals
- Enhanced flexibility in retry logic based on chunk state

### Changed

- Updated parameter naming for better code clarity

## [0.1.4] - 2024

### Fixed

- Various bug fixes and improvements

## [0.1.3] - 2024

### Added

- Comprehensive test suite with Jest
- Code coverage reporting
- ESLint and Prettier configuration
- Husky pre-commit hooks
- Commitlint for conventional commits

### Changed

- Improved project structure and tooling

## [0.1.2] - 2024

### Fixed

- Minor bug fixes and improvements

## [0.1.1] - 2024

### Fixed

- TypeScript definitions improvements
- Documentation updates

## [0.1.0] - 2024

### Added

- Initial release of resumable-uploads
- TypeScript support
- Fork of original resumable.js with modern tooling
- Core upload functionality with chunking
- File validation (type, size)
- Retry logic with configurable options
- Multiple simultaneous uploads
- Browser compatibility for HTML5 File API
- Comprehensive documentation
- MIT License
